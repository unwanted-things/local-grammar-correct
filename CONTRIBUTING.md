# Contributing to Local Grammar Correct 🎯

This project is all about having fun with grammar! All code is AI-generated, so don't stress about perfection. Just vibe with it! 🎵

1. Fork the repo
2. Make your changes (with AI help!)
3. Submit a PR
4. Have fun! 🎮
